Typography

This page showcases the typography styles defined in our Sass variables and components.

Display headings are used for large, impactful text elements like hero sections and major page headers.

Body Text

Body text styles for general content.

This is large body text. Used for important paragraphs or when you need slightly larger text for emphasis.

.body-large {
  font-size: $body-large-size; // 18px
  line-height: $body-large-line-height; // 27px
  font-weight: $font-weight-normal; // 400
}

This is default body text. The standard text size used for most content throughout the application.

.body-default {
  font-size: $body-default-size; // 16px
  line-height: $body-default-line-height; // 24px
  font-weight: $font-weight-normal; // 400
}

This is small body text. Used for less important information, captions, footnotes, etc.

.body-small {
  font-size: $body-small-size; // 14px
  line-height: $body-small-line-height; // 21px
  font-weight: $font-weight-normal; // 400
}

This is lead text. Used for introductory paragraphs or to highlight important information.

.lead {
  font-size: $lead-font-size; // 21px
  line-height: $lead-line-height; // 32px
  font-weight: $font-weight-normal; // 400
}
